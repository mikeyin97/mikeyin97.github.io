My Work Experience

Researcher, May 2016 to August 2016

During the summer, I worked at the Intelligent Sensory Microsystems Laboratory at the University of Toronto. In collaboration with a startup called MyAbilities Inc., we performed research and proof-of-concept development of a wearable biomedical device for injury and performance testing.

Some of my responsibilities were to

  • Collect and analyze research data in real-time using various frequency processing and machine learning algorithms, written primarily in Arduino, Python, and MATLAB.
  • Store motion sensor data in both xml format as well as on a local server using node.js
  • Help animate a 3D model of the human head and torso on a web browser using this stored data. This was done using HTML, CSS, and Javascript.
